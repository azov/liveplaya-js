BRCMap JavaScript Library

Download v0.0.0 | Other releases | Source code

This is a JavaScript Library for rendering live interactive maps of Black Rock City, the home of Burning Man festival. It supports offline static maps that work without a server or internet connection as well as online maps that show real-time data, e.g. art car locations.
